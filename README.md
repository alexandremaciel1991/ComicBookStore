Projeto feito em NextJs

## Configuração

Existem 3 variaveis para se configurar.
Elas podem se obtidas no [Portal da Marvel](https://developer.marvel.com/), conforme instruções do mesmo.

```bash
TS=
APIKEY=
HASH=
```

## Rodando

```bash
yarn install
yarn dev
```
